People around the world using FluidDB

Learn to add yourself by following the guide.

Guide

Here is a guide how to add yourself on the map. All you need is a FluidDB account and some way to set some tag values; the FluidDB Explorer offers one way to do that.

Conventions

To get yourself on the map, you have to do two things, both of which are achieved by tagging object in FluidDB.

  1. Give values for your latitude and longitude
  2. Tell the people application that you exist

Let's assume that your FluidDB username is foobar. First you need to set your latitude and longitude by attaching tags foobar/people/latitude and foobar/people/longitude to the appropriate numeric values. You attach these tags to the FluidDB object that corresponds to you, which if your username is foobar is the object with the about tag Object for the user named foobar. An example of a user who has done this is paparent; you can see his tagged object here in a modern browser.

To let the people application know that you exist, you need to tag the object with about tag collection:peopleapp (62397fb0-6f96-404a-a8fa-ee3758cfa7f2) with the tag foobar/peopleapp. You can see this object in most modern web broswers by clicking here.

Once you've done these two things, you should be live in the people application.

Methods

FluidDB Explorer

Please follow these quick steps:

  1. Browse to the explorer with your username at the end of the URL, e.g. http://explorer.fluidinfo.com/fluiddb/paparent for username paparent
  2. Click on the Log in with your FluidDB ID and fill-in the form to log in the Explorer.
  3. On the left-hand of the application, there's the namespace tree. Right click on your username, create a tag named peopleapp.
  4. Create a namespace named people with two tags in it: latitude and longitude.
  5. Find your user object via the Query field on top right; enter the query fluiddb/about="Object for the user named foobar" (replace foobar with your username).
  6. You should find the object so double click it to open the object window.
  7. Add your latitude and longitude tags on your user object.
  8. Find the collection object with the query field again; enter the query fluiddb/about="collection:peopleapp".
  9. Add your peopleapp tag to that collection
  10. Refresh this page and enjoy!
